Description détaillée

Cette permission autorise les agents certifiés de l'état civil à valider les demandes numériques d'actes de naissance soumises par les citoyens via le portail E-Gov. Elle inclut la vérification des pièces justificatives, la signature électronique du document et l'envoi de la notification de disponibilité au demandeur.
